# CSS引号

属性选择器或属性值用双引号（””），而不是单引号（”）括起来。

URI值（url()）不要使用引号。

不推荐：

    @import url('//cdn.com/foundation.css');

    html {
    font-family: 'open sans', arial, sans-serif;
    }

    body:after {
    content: 'pause';
    }
