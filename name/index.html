# 命名规范

# HTML/CSS命名

确保文件命名总是以字母开头而不是数字，且页面及样式文件以小驼峰的方式命名，组件的命名以大驼峰的方式，如：

    <!-- HTML -->
    userManage.vue
    carManage.vue
    carOperate.vue

    <!-- SASS -->
    userManage.scss
    carManage.scss
    carOperate.scss

    <!-- Components -->
    Header.vue
    Footer.vue
    Sidebar.vue

# ClassName命名
